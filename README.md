# 기숙사 관리 프로그램 매뉴얼

한성과학고 기숙사 관리 프로그램 통합 사용 매뉴얼 (PWA)

## 접속 주소
https://사용자명.github.io/저장소명/

## 포함 프로그램
- 자습실 출석부
- 기숙사 아침 점호
- 심야 자습 출석부
- 전자기기 관리
- 퇴실 체크아웃

## 업로드 파일 목록
| 파일 | 역할 |
|------|------|
| index.html | 매뉴얼 본체 |
| manual-manifest.json | PWA 앱 설정 |
| manual-sw.js | 오프라인 캐시 |
| manual-icon-192.png | 앱 아이콘 (192px) |
| manual-icon-512.png | 앱 아이콘 (512px) |
