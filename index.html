들어가며
기숙사 관리 프로그램은 남·여 기숙사의 학생 출결과 생활지도를 정확하고 빠르게 처리하도록 만든 도구 모음입니다.

태블릿 화면을 손가락으로 터치하는 것만으로 출결을 기록하고, 그 결과를 엑셀 파일이나 카카오톡 명단으로 바로 내보낼 수 있습니다. 모든 기록은 기기에 자동 저장되며, 구글 드라이브 자동 백업으로 한 번 더 안전하게 보관됩니다. 처음 한 번만 설정해 두면, 매일의 사용은 화면을 몇 번 누르는 것으로 끝납니다.
이 매뉴얼 앱의 구성

공통안내 탭에는 모든 프로그램에 공통으로 해당하는 내용(설치·업데이트·백업·메모·주의사항)을 담았습니다. 처음 한 번만 읽어 두시면 이후 각 프로그램을 훨씬 쉽게 사용할 수 있습니다.

프로그램 탭에서는 6개 프로그램 중 하나를 선택해 실제 사용 화면처럼 기능을 차례대로 확인할 수 있습니다.

📦 여섯 가지 프로그램
  • 1자습실 출석부 — 야간 자습 1·2·3교시의 상세 출결을 좌석 배치도 위에서 관리
  • 2아침 점호 — 학번을 입력해 출석을 체크하고 지각·불참을 자동 분류
  • 3심야 자습 출석부 — 좌석을 눌러 출석/부재만 빠르게 체크
  • 4전자기기 관리 — 핸드폰·태블릿·노트북 제출 여부를 학년별로 체크
  • 5퇴실 체크아웃 — 호실별 상·벌점과 정리정돈 등급, 귀가·퇴사 명단을 관리
  • 6기숙사 관리일지 — 근무일지·상벌점·상담·학사일정·관심학생을 한곳에서 관리

1-1. PWA 방식이란?
🌐
웹사이트인데 앱처럼

이 프로그램들은 모두 PWA(Progressive Web App) 방식입니다. 쉽게 말해 웹사이트인데 앱처럼 쓸 수 있는 프로그램입니다. 앱 스토어에서 내려받지 않아도 되고, 인터넷 주소(링크)만 알면 바로 접속해 사용할 수 있습니다. 한 번 홈 화면에 설치하면 일반 앱처럼 아이콘을 눌러 실행합니다.

🔗
접속 방법
담당 선생님이 카카오톡으로 각 프로그램의 링크를 공유해 드립니다. 받은 링크를 누른 뒤, 아래 1-2의 안내에 따라 크롬 브라우저로 열어 홈 화면에 추가하면 앱처럼 사용할 수 있습니다.
1-2. 앱처럼 설치하기
📱
안드로이드 태블릿 (갤럭시 탭)
  1. 1카카오톡에서 공유받은 링크를 누릅니다. (카카오톡 안의 간이 브라우저가 열립니다.)
  2. 2크롬에서 열리면 주소창 오른쪽 끝의 점 세 개 메뉴(⋮)를 누릅니다.
  3. 3[홈 화면에 추가] → [설치]를 누릅니다.
  4. 4바탕화면에 생긴 아이콘을 누르면 앱처럼 전체 화면으로 실행됩니다.
⚠️
반드시 크롬으로 열어 주세요
카카오톡 안의 간이 브라우저에서는 홈 화면 추가가 되지 않습니다. 화면 우측 하단의 [다른 브라우저로 열기] 또는 [Chrome으로 열기]를 눌러 크롬으로 다시 열어 주세요.
💻
개인 PC (크롬 브라우저)
  1. 1크롬 주소창에 프로그램 링크를 입력해 접속합니다.
  2. 2주소창 오른쪽 끝의 설치 아이콘(⤓ 또는 모니터 모양)을 누르거나, 점 세 개 메뉴(⋮)에서 설치를 선택합니다.
  3. 3설치하면 주소창 없이 독립된 창으로 실행됩니다. (설치하지 않고 즐겨찾기로 사용해도 됩니다.)
1-3. 자동 업데이트와 화면 특징
  • 자동 업데이트 — 프로그램이 업데이트되면 선생님은 아무것도 하지 않아도 됩니다. 앱 종료 후 실행하거나 화면을 새로고침하면 최신 내용이 자동으로 반영됩니다.
  • 전용 앱 화면 — 아이콘으로 실행하면 주소창 없는 전체 화면으로 열립니다. 브라우저 뒤로가기 대신 프로그램 안의 버튼으로 이동하세요.
  • 기기 내 자동 저장 — 등록한 명단·설정과 매일의 기록은 그 기기 안에 자동 저장됩니다.
🔄
최신 화면이 바로 보이지 않을 때
앱을 완전히 종료(최근 앱 목록에서 위로 밀어 닫기)한 뒤 다시 실행하거나, 브라우저 캐시를 삭제하고 다시 접속하면 최신 버전으로 갱신됩니다.
1-4. 구글 드라이브 자동 백업
☁️
한 번 연결하면 끝

기기에만 저장되던 백업을, 선생님 본인의 구글 드라이브에 자동으로 올려 두는 기능입니다. 태블릿이 고장 나거나 초기화되어도 드라이브의 백업으로 모든 기록을 되살릴 수 있습니다.

이런 점이 좋습니다
  • 내 계정, 내 드라이브에 저장 — 선생님 본인의 구글 계정으로 연결되고, 백업은 본인 드라이브에만 저장됩니다. 다른 사람은 볼 수 없어 개인정보가 보호됩니다.
  • 프로그램별 폴더 자동 정리 — 백업 폴더는 '프로그램이름_백업' 형태로 자동 생성됩니다. 폴더 안에는 항상 최신 파일 1개와 날짜별 파일이 보관되고, 정해 둔 기간이 지난 파일은 자동으로 정리됩니다.
  • 한 번 켜 두면 끝 — '앱을 켤 때 자동 백업'을 켜 두면, 이후에는 프로그램을 실행할 때마다 알아서 백업됩니다.
✅
선생님이 하실 일은 두 가지뿐입니다
① 담당 선생님에게 '클라이언트 ID' 받기 (최초 1회, 모든 프로그램 공용)
② 사용하는 프로그램마다 한 번씩 '구글 연결' 하기 (각 1~2분)
구글 클라우드 콘솔 설정 같은 어려운 준비는 모두 끝나 있으므로, 선생님은 신경 쓰실 필요가 없습니다.

모든 프로그램의 [⚙️ 설정] 안에는 아래와 같은 구글 드라이브 자동 백업 영역이 들어 있습니다. 모양과 사용법은 모든 프로그램이 동일합니다.

연결 방법 (프로그램마다 한 번씩)
  1. 1프로그램 상단의 [⚙️ 설정]을 열고, 구글 드라이브 자동 백업 영역을 펼칩니다. ([드라이브 연결 설정 열기])
  2. 2담당 선생님에게 받은 OAuth 클라이언트 ID(끝이 .apps.googleusercontent.com인 긴 문자열)를 첫 번째 입력칸에 그대로 붙여넣습니다.
  3. 3[🔗 구글 연결] 버튼을 누릅니다. 새 창이 열리면 본인 Gmail 계정으로 로그인합니다.
  4. 4"이 앱이 Google Drive에 접근하도록 허용" 화면이 나오면 [허용/계속]을 누릅니다. (선생님이 만든 파일만 다루는 권한이라 안전합니다.)
  5. 5[앱을 켤 때 자동으로 드라이브에 백업] 체크박스를 켭니다(ON). 이제 연결이 완료되었습니다.
  6. 6필요하면 날짜별 파일 보관 일수(기본 30일)를 정합니다. 이 기간이 지난 날짜별 파일은 자동으로 정리됩니다.

PROGRAM 01

📋 자습실 출석부

야간 자습 1·2·3교시 상세 출결 관리
좌석 배치도 위에서 색상으로 구분, 부재자 카카오톡 전달

야간 자습 1·2·3교시의 상세 출결을 좌석 배치도 위에서 관리하는 프로그램입니다. 조퇴·외출·상담·도서관 등 다양한 사유를 색으로 구분해 기록하고, 부재자 명단을 카카오톡으로 바로 전달할 수 있습니다.

2-1. 최초 준비 — 자습실 이름과 좌석 배치도

처음 사용할 때 한 번만 준비하면 됩니다. 먼저 화면 상단 가운데의 큰 글씨(예: "1학년 자습실 ✏️")를 눌러 자습실 이름을 학년에 맞게 정하세요. 그다음 [⚙️ 설정]에서 좌석 배치도 엑셀을 한 번 등록합니다.

  • 📥 배치도 엑셀 불러오기 — 좌석 배열·교사석·좌석 번호가 담긴 엑셀을 등록합니다. 한 번 등록하면 자리 변경 전까지 그대로 유지됩니다.
🔥
학년마다 이름을 다르게 정하세요
입력한 이름은 백업 파일과 구글 드라이브 폴더 이름에 그대로 쓰입니다. 학년이 다른데 이름이 같으면 서로의 백업을 덮어쓸 수 있습니다.
2-2. 매일 — 당일 출석부 연동

자습 시작 전, 상단의 날짜 영역(출석부 연동)을 통해 그날의 당일 출석부 엑셀(귀가·외출·교내활동·도서관 명단)을 연동합니다. 그러면 해당 학생의 좌석이 자동으로 색과 상태가 적용됩니다.

💡
날짜 자동 인식

아침 등교 후 호실별 상점·벌점과 정리정돈 상태를 기록하고, 귀가·퇴사·미입사 명단을 한 화면에서 관리하는 프로그램입니다. 담당하는 두 개 층을 탭으로 전환하며 사용합니다.

6-1. 최초 설정 — 담당 층과 명단

6-2. 호실 상·벌점과 정리정돈 상태 기록

호실 카드 제목(예: "201호")을 누르면 호실 창이 열립니다. 호실 전체 벌점(호실정리불량·슬리퍼 미정리·미소등 등 공통)과 정리정돈 우수 등급(상·중·하)을 선택합니다. 개별 학생을 누르면 그 학생에게만 개인 벌점을 줄 수 있습니다.

항목내용

정리정돈 우수 등급상(3점)·중(2점)·하(1점) 선택 — 상점 누적
개인 벌점개별 학생을 눌러 그 학생에게만 적용

상단 [📊 오늘 집계]로 그날 결과를 정리해 복사할 수 있고, [🏅 상점 집계]로 특정 기간 내 정리정돈 우수 등급(상 3·중 2·하 1점)을 합산해 호실 순위를 매길 수 있습니다.

🏅
이 방법으로 2주에 한 번 정리정돈 우수 호실들을 선정해 단체 상점(3점)을 주시면 됩니다.

근무일지·상벌점·상담·학사일정·관심학생
생활지도 기록을 한곳에서 종합 관리

근무일지·상벌점·상담·학사일정·관심학생 등 기숙사 생활지도 기록을 한곳에 모으는 종합 관리 프로그램입니다. 전교생을 학년별·호실별로 보면서 학생별 종합 정보를 관리합니다.

7-1. 화면 구성
  • 학년별 / 호실별 보기 — 오른쪽 위 토글로 전교생을 학년 순 또는 호실 순으로 볼 수 있습니다.
  • 학생 카드 — 카드에 학번·이름이 표시되고, 누르면 그 학생의 상세 정보가 열립니다.
  • 상단 기능 버튼 — ⚙️ 설정 · 📝 메모 · 📔 근무일지 · 📊 당일상벌점 · ✏️ 일괄입력 · 📅 학사일정 · ⭐ 관심학생
7-2. 학생 상세 정보와 종합카드

학생 카드를 누르면 학생 상세 창이 열립니다. 연락처·보호자·특이사항을 적고 상벌점·상담 내역을 기록할 수 있으며, 학생을 일반·관심·위기로 분류할 수 있습니다.

일반
⭐ 관심
🚨 위기

오른쪽 위 [📋 종합카드]를 누르면 한 학생의 모든 정보(사진·연락처·상벌점·상담 내역)를 한 장으로 모아 보고 인쇄할 수 있습니다.

7-3. 상벌점 — 개별·일괄 입력

학생 상세의 상벌점 탭에서 개별로 줄 수도 있고, 상단 [✏️ 일괄입력]으로 여러 학생에게 같은 항목을 한 번에 줄 수도 있습니다.

  1. 1상벌점 항목과 사유를 선택합니다.
  2. 2학번을 입력하고 [추가]를 눌러 대상 학생을 모읍니다. (여러 명 가능)
  3. 3[일괄 적용]을 누르면 모든 학생에게 한 번에 기록됩니다.
7-4. 당일 상벌점과 관심학생
  • 📊 당일상벌점 — 근무 시간(예: 15:00~익일 09:00) 동안 받은 상벌점을 모아 보여 줍니다.
  • ⭐ 관심학생 — 관심·위기로 분류한 학생만 모아 보여 줍니다.
7-5. 근무일지와 학사일정

[📔 근무일지]는 날짜별 일지와 당일 상담 요약을 적고 확인하는 공간입니다. [📅 학사일정]은 학기별 달력에 학사 일정과 할일을 추가·수정하며 관리합니다.

📅
한성과학고 1년간 학사 일정이 그대로 이식되어 있으며, 날짜를 탭하여 일정을 수정·추가할 수 있습니다. 날짜를 탭하여 할일을 등록해 놓으면 해당일에 알림 팝업창이 뜹니다.
7-6. 설정 · 데이터

[⚙️ 설정]에서 당일 귀가/퇴사 명단 연동, 담당 사감 이름, 전체 백업/복원, 구글 드라이브 자동 백업(공통안내 1-4 참고)을 설정합니다.

  • 🏠 당일 귀가/퇴사 명단 연동 — 귀가외출교내활동 엑셀 또는 기숙사 일지 엑셀을 불러오면 해당 학생 카드가 옅은 빨강으로 표시됩니다.
  • 💾 전체 백업 / 복원 — 명단·연락처·사진·모든 기록을 파일 하나로 저장하고 복원합니다.
  • ☁️ 구글 드라이브 자동 백업 — 공통안내 1-4 안내대로 연결하면 자동 백업됩니다.
