들어가며
태하관 기숙사 관리 프로그램은 남·여 기숙사의 학생 출결과 생활지도를 사감 선생님 여러 분이 함께, 더 정확하고 빠르게 처리하기 위해 만든 프로그램 모음입니다.

태블릿 화면을 손가락으로 터치하는 것만으로 출결을 기록하고, 그 결과를 엑셀 파일과 카카오톡 명단으로 바로 내보낼 수 있도록 설계되었습니다.
주요 특징
  • 설치가 필요 없습니다. 링크에 접속만 하면 바로 사용, 원하면 홈 화면에 설치 가능
  • 태블릿에 최적화. 갤럭시 탭 S9 FE+ 세로 화면 기준, PC 크롬에서도 동일하게 사용 가능
  • 데이터 자동 저장. 명단·배치도·설정은 한 번 등록하면 다음에 켜도 유지
  • 엑셀 백업·복원 지원. 기기 교체·초기화 시 백업 파일로 전체 복원 가능
  • 자동 업데이트. 선생님이 별도 설치 없이 다음 실행 시 자동 반영

1. PWA 방식이란?
🌐
PWA 방식 설명

이 프로그램들은 모두 PWA(Progressive Web App) 방식으로 제공됩니다. 웹사이트인데 앱처럼 쓸 수 있는 프로그램입니다.

🔗
담당 선생님이 카카오톡으로 각 프로그램의 링크(인터넷 주소)를 공유해 드립니다. 받은 링크를 누른 뒤 크롬 브라우저로 열어 홈 화면에 추가하면 앱처럼 사용할 수 있습니다.
2. 앱처럼 설치하기
📱
안드로이드 태블릿 (갤럭시 탭)
  1. 1카카오톡에서 공유받은 링크를 누릅니다.
  2. 2[다른 브라우저로 열기] 또는 [Chrome으로 열기]를 눌러 반드시 크롬으로 엽니다.
  3. 3주소창 오른쪽 끝 점 세 개 메뉴(⋮)를 누릅니다.
  4. 4하단 메뉴에서 [홈 화면에 추가]를 누릅니다.
  5. 5[설치]를 누릅니다.
  6. 6바탕화면 아이콘을 누르면 전체 화면으로 실행됩니다.
⚠️
카카오톡 안의 간이 브라우저에서는 홈 화면 추가가 되지 않습니다. 반드시 크롬으로 열어 주세요.
💻
개인 PC (크롬 브라우저)
  1. 1크롬 주소창에 프로그램 링크를 복사해 접속합니다.
  2. 2주소창 오른쪽 끝 설치 아이콘(⤓)을 누르거나, 점 세 개 메뉴 → [프로그램명/설치]를 선택합니다.
  3. 3설치하면 주소창 없이 독립된 창으로 실행됩니다.
💡
설치하지 않고 즐겨찾기로 사용해도 됩니다. PWA로 설치하면 더 깔끔하게 사용할 수 있습니다.
3. PWA 주요 특징
  • 업데이트 자동 적용 — 사용자는 아무것도 하지 않아도 됩니다. 다음 실행 시 자동 반영
  • 전용 앱 화면 — 주소창이 없는 전체 화면으로 열립니다. 프로그램 안의 버튼으로 이동하세요
  • 일부 오프라인 사용 — 최초 접속 시 인터넷 필요, 이후 일부 기능은 오프라인 사용 가능
  • 기기 내 저장 — 명단·배치도·설정과 매일의 기록은 그 기기 안에 보관됩니다
🔄
업데이트가 바로 반영되지 않을 때
앱을 완전히 종료한 뒤 다시 실행하세요. 최근 앱 목록에서 해당 앱을 위로 밀어 닫아 주세요. 홈 버튼만 누르면 백그라운드에서 계속 실행됩니다.
4. 공통 주의사항
⚠️
데이터 보호 — 반드시 지켜 주세요
브라우저 데이터(캐시·사이트 데이터)를 삭제하면 기기 내 저장 데이터가 함께 지워집니다. 정기 백업을 생활화하세요.
🔥
학년마다 제목(타이틀)을 다르게 설정하세요
제목이 같으면 백업 파일명이 똑같아져 기존 데이터가 덮어써져 영구 삭제될 수 있습니다.
📁
안드로이드 다운로드 폴더 안내
파일은 자동으로 [다운로드] 폴더에 저장됩니다. [내 파일] 앱에서 전용 폴더(예: '태하관백업')를 만들어 정기적으로 정리하세요.

자습실 출석부의 단순 버전

사용 방법
  1. 1[🔄 배치도] 버튼으로 좌석 배치도 엑셀을 처음 한 번 등록합니다. (이후 계속 유지)
  2. 2상단 제목을 눌러 자습실 이름을 학년에 맞게 바꿉니다.
  3. 3출석한 학생의 좌석을 누르면 초록색(출석)으로 바뀝니다. 다시 누르면 흰색(부재)으로 돌아갑니다.
  4. 4상단에 총원 · 출석 · 부재 인원이 실시간 집계됩니다.

  2. 2상단 [1학년] [2학년] [3학년] 탭으로 학년을 전환하며 체크합니다.
  3. 3미제출 기기만 눌러 빨강으로 표시합니다. 나머지는 그대로 두면 제출 완료 처리됩니다.

귀가·퇴사·미입사 학생은 음영 처리되어 호실 전체 벌점 입력 시 자동으로 제외됩니다.
귀가·퇴사·미입사 관리

[🚪 귀가·퇴사] 버튼에서 그날 체크 제외 학생을 관리합니다.

  • 명단 텍스트 붙여넣기 → "귀가·퇴사·명령퇴사·미입사" 키워드를 자동 인식해 학번/이름으로 매칭
  • 상단 일지 연동 버튼에서 '귀가외출교내활동' 엑셀을 연동하면 자동 반영, 이후 개별 수정 가능
집계 & 백업
  • 📊 오늘 집계 — 오늘 체크아웃 결과 정리, [📋 복사]로 내용 복사
  • 🏅 상점 집계 — 기간(시작일~종료일) 내 정리정돈 우수 등급 합산 순위
  • 📝 당일 특이사항 메모 — 전달 사항 자유롭게 입력, 자동 저장
  • 💾 전체 백업(1개) / 📂 복원 — 모든 날짜 상·벌점·등급·메모를 파일 하나로 저장
  • 🗓️ 기록 달력 — 파란 점이 있는 날은 기록 있는 날. 날짜 누르면 그날 기록으로 이동
⏰
자동 초기화 — 새 날이 시작되면(늦어도 오전 11시 이전) 당일 빈 시트로 자동 전환됩니다. 전날 기록은 영구 보관됩니다.

✅ 매일 체크리스트

매일 반드시 해야 할 작업 요약

자습실 출석부 / 심야 자습
  1. 1자습 시작 전: 당일 출석부 엑셀 연동
  2. 2자습 중: 좌석 상태 체크
  3. 33교시 후: [📤 3교시 출결 내보내기]로 부재자 명단 전송
  4. 4종료 후: [🗓️ 과거 기록] → [📥 백업] 실행
아침 점호
  1. 1당일 명단 등록
  2. 2데이터 동기화
  3. 3[📊 당일 점호 종료]로 집계
  4. 4백업 실행
전자기기 관리
  1. 1일지 연동
  2. 2학년별 미제출만 빨강으로 체크
  3. 3미제출 현황 확인
  4. 4백업 실행
퇴실 체크아웃
  1. 1일지 연동 / 제외 학생 등록
  2. 2호실별 상·벌점·등급 입력
  3. 3[📊 오늘 집계] 확인
  4. 4메모 작성
공통 주간 작업
  • 주 1회 다운로드 폴더의 파일을 전용 백업 폴더로 정리
  • 백업 파일 안전한 장소에 보관 (개인 드라이브 또는 USB)
